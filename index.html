Getting started (drylab)¶

Set up your computer for drylab development¶

  1. First thing you’ll want to do is get an account on TSCC.
Email Jim Hayes and ask to be added to the groups yeo-group and scrm-group.
  1. If you don’t have a Github account yet, go make one.
  2. Ask to get added to the YeoLab group on Github.
  3. Install hub. If you have a Mac, install via brew.
  4. Install the Anaconda Python Distribution to your laptop.
  5. Install PyCharm (Community edition is fine).
