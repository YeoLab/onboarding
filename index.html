Getting started (drylab)¶

Set up your computer for drylab development¶

  1. First thing you’ll want to do is get an account on TSCC. Email Jim Hayes and ask to be added to the groups yeo-group and scrm-group.
  2. If you don’t have a Github account yet, go make one.
  3. Ask to get added to the YeoLab group on Github.
  1. Install the Anaconda Python Distribution to your laptop. (Get Python 2.7 for now)
  2. Install PyCharm (Community edition is fine).
